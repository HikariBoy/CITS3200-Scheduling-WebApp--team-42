## How to Use

### Step 1: Create the Unit
1. Log in as Unit Coordinator
2. Go to "Create Unit"
3. Fill in:
   - **Unit Code:** MECH1100 (or MECH2401, MECH3401)
   - **Unit Name:** Introduction to Mechanical Engineering (or appropriate name)
   - **Year:** 2025
   - **Semester:** Semester 1
   - **Start Date:** 24/02/2025 (Week 1)
   - **End Date:** 23/06/2025 (Week 18)

### Step 2: Upload Facilitators
1. Click "Upload Facilitators CSV"
2. Select the corresponding facilitators file:
   - `mech1100_facilitators.csv`
   - `mech2401_facilitators.csv`
   - `mech3401_facilitators.csv`

### Step 3: Upload Sessions
1. Click "Upload Sessions CSV"
2. Select the corresponding sessions file:
   - `mech1100_sessions.csv`
   - `mech2401_sessions.csv`
   - `mech3401_sessions.csv`

### Step 4: Test Auto-Assignment
1. Have facilitators declare their skills
2. Run auto-assignment
3. Download schedule report
